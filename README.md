# Go Server Application

This repository contains a simple Go server application that demonstrates basic HTTP handling and JSON responses.

## Overview

The project includes a basic HTTP server written in Go that responds to API requests. It serves as a foundation for building more complex web applications.

## Features

- Simple HTTP server implementation
- JSON response handling
- Basic routing

## Requirements

- Go 1.16 or higher

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Run the server with `go run server.go`

## Usage

After starting the server, you can access it at `http://localhost:8080`

## Files

- `server.go` - Main server application
- `README.md` - Project documentation
